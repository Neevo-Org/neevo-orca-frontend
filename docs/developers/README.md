# Developer Docs

Developer documentation is intended for engineers and coding agents working in this repository.

Folders represent groups. Individual `.md` files represent pages.
The file name defines the page name shown in the side menu.
Page file names should use capitalization intended for display.

Current pages:

- [`Rules.md`](Rules.md): canonical development rules for structure, routing, UI, and workflow
- [`Data Modes.md`](<Data Modes.md>): mock and API development mode strategy
- [`Structure.md`](Structure.md): frontend structure, ownership, and development rules
